# obsidian-work

Obsidian workspace for work-related notes and documentation through ocool.

## About

This repository contains an Obsidian vault configured for work use. Obsidian is a powerful knowledge base that works on top of a local folder of plain text Markdown files.

## Structure

```
.
├── .obsidian/          # Obsidian configuration files
├── notes/              # Your main notes and documentation
├── templates/          # Note templates
├── attachments/        # Images and file attachments
└── README.md
```

## Getting Started

1. Install [Obsidian](https://obsidian.md/)
2. Clone this repository
3. Open the repository folder as a vault in Obsidian
4. Start taking notes!

## Features

- Pre-configured workspace layout
- Organized folder structure
- Note templates for consistency
- Automatic link updates
- Attachment management

## Configuration

The vault includes sensible defaults in `.obsidian/`:
- Live preview mode enabled
- Automatic link updates
- Attachments stored in `attachments/` folder

## Usage

- Create new notes in the `notes/` folder
- Use templates from the `templates/` folder for consistent formatting
- Link notes together using `[[double brackets]]`
- Add images and files to the `attachments/` folder
