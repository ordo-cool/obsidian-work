# Welcome to Obsidian Work

This is your Obsidian workspace for work-related notes and documentation.

## Getting Started

This vault is organized into the following folders:

- **notes/** - Your main notes and documentation
- **templates/** - Note templates for consistent formatting
- **attachments/** - Images and other file attachments

## Tips

- Use `[[double brackets]]` to create links between notes
- Press `Ctrl/Cmd + P` to open the command palette
- Create new notes by clicking on a link to a non-existent note

## Configuration

The workspace includes basic configuration in `.obsidian/`:
- **app.json** - General application settings
- **appearance.json** - Theme and visual settings
- **workspace.json** - Layout and open files

Happy note-taking!
